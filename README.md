# SSINDEX Landing Page

A bespoke single-page pitch prepared for Stakeholders Sustainability Index (SSINDEX), exhibitor at eMerge Americas 2026, booth SS96. The page carries the SSINDEX-voice pitch from the fit analysis: where AI fits in the SSINDEX methodology, and where it does not. Designed to feel native to ssindex.com (teal primary, navy body, Roboto Slab display, Roboto body, stat-block rhythm) so the prospect reads it as their own marketing team's work with Eduba's substance inside.

Intended URL path: /for/ssindex

Files:
- `index.html` single static page, no JavaScript, links the site-native Google Fonts used by ssindex.com.
- `styles.css` all styling, mobile-first, tested at 375px.
- `logo.png` copied from the brand pack assets.
